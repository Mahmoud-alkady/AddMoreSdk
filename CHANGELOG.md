## [0.1.0] - 2024-11-10

### Added
- Initial release of AdMoreSdk
- iOS 12.0+ support
- Swift Package Manager integration
- Basic data collection functionality
- Privacy-first GDPR/CCPA compliant data collection
- Comprehensive device and user data collection (20+ categories)
- Binary framework distribution via XCFramework

### Features
- Simple two-method integration
- Event tracking capabilities
- Banner ad display support
- Full-screen ad presentation
- Core Data caching support
